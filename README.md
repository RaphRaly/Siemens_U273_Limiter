# U73 Limiter / Siemens U273

Public beta download and source repository for the U73/U273 vintage gain-cell limiter plugin.

## Download Beta_1.0

- [Download Windows VST3 Beta_1.0](u273-plugin/releases/Beta_1.0/U273_Limiter_Beta_1.0_Windows_VST3.zip)
- Platform: Windows x64
- Format: VST3
- Build: Release
- SHA256: `24C53D82628706E741EE9A98FA24657DC966E42D4EA02F72864C16F70CE3061F`

## Install

1. Download and unzip the Beta_1.0 archive.
2. Copy the extracted `U273 Limiter.vst3` folder to:

```text
C:\Program Files\Common Files\VST3
```

3. Rescan plugins in your DAW.

The plugin currently appears to hosts as `U273 Limiter`; the current GUI faceplate is marked `U73 Limiter`.
